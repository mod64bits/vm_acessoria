# VM Acessoria

Sistema Simples de Orçamento

## 🔧 Funções

- Multe Empresa
- Cadastro de Equipamentos
- Cadastro de Mão de Obra
- Orçamento 
- Orçamnetos com Preços Difiretntes
- Multe Empresas para gerar orçamento com CNPJ diferente
